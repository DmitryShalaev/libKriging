# `leaveOneOut`

Compute Leave-One-Out


## Description

Compute the leave-One-Out error of a model given in `object` .


## Usage

```r
leaveOneOut(object, ...)
```


## Arguments

Argument      |Description
------------- |----------------
`object`     |     An object representing a fitted model.
`...`     |     Ignored.


## Value

The Leave-One-Out sum of squares.
