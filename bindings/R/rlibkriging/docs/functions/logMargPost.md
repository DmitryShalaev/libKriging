# `logMargPost`

Compute log-Marginal Posterior


## Description

Compute the log-Marginal Posterior of a model given in
 `object` .


## Usage

```r
logMargPost(object, ...)
```


## Arguments

Argument      |Description
------------- |----------------
`object`     |     An object representing a fitted model.
`...`     |     Ignored.


## Value

The log-marginal posterior.
