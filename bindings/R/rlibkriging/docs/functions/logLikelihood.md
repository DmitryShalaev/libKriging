# `logLikelihood`

Compute Log-Likelihood


## Description

Compute the log-Likelihood of a model given in `object` .


## Usage

```r
logLikelihood(object, ...)
```


## Arguments

Argument      |Description
------------- |----------------
`object`     |     An object representing a fitted model.
`...`     |     Ignored.


## Value

The log-likelihood.
